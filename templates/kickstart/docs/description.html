Bootstrap 3 based template for Subrion CMS

Features:

  • Many configuration options
  • Landing page (can be turned off)
  • Mobile ready
  • Compatible with plugins: blog, portfolio, testimonials

Heads up This is a developer template and has only base styles. More updates coming soon.
